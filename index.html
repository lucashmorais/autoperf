What is autoperf?

autoperf is a little open-source program for displaying Linux perf code profiling information in nicely formatted HTML reports, clearly highlighting hot functions and code sections. It should be useful for anyone tinkering with simple C or C++ performance profiling on Linux.

Quickstart

After extracting the project tarball and profiling your binary file with perf record as usual, do:

$ cd [pathtoautoperf]
$ ./autoperf -i [somepath]/perf.data

... and check the result on the template folder!

Documentation

For further information on program usage, check our video overview at https://youtu.be/Toub5SpKUB8 and our project wiki at https://github.com/lucashmorais/autoperf/wiki.

Package dependencies

  • exuberant-ctags
  • linux-tools-generic
  • openjdk

Tested Kernel versions

  • Kernel 3.13
  • Kernel 3.16

Known issues

  • Source code that may be subject to preprocessor modification may not be correctly annotated by autoperf.
